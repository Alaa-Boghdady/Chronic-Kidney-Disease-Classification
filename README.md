# Chronic Kidney Disease Classification 🩺

## 📌 Project Overview
This project applies Machine Learning to healthcare by predicting Chronic Kidney Disease (CKD) in patients. Using a dataset of clinical features, the model classifies patients into "CKD" or "Not-CKD" groups.

## 🛠️ Technical Workflow
* **Exploratory Data Analysis (EDA):** Analyzing medical features like Hemoglobin, Blood Pressure, and Albumin levels.
* **Missing Value Imputation:** Handling incomplete medical records using statistical techniques.
* **Modeling:** Comparing **Random Forest** and **Decision Tree** classifiers for optimal performance.
* **Evaluation:** Prioritizing **Recall** to minimize False Negatives in diagnosis.

## 💻 Tech Stack
* **Language:** Python (Google Colab)
* **Libraries:** Scikit-learn, Pandas, Seaborn.
